Bootstrap

O que é?

Bootstrap é uma ferramenta gratuita para desenvolvimento de HTML, CSS e JS. É possível criar rapidamente prototipos de suas ideias ou desenvolva aplicações completas com as variaveis e mixins SASS, sistema de grid responsivo, componentes pré-construidos extensos e plugins desenvolvidos em jQuery.
